# Tweeter Project

Tweeter is a simple, single-page Twitter clone.

!["Screenshot of URLs page"](./docs/1.gif)

!["Screenshot of Edit page"](./docs/2.gif)

## Dependencies

- Node.js
- Express
- Chance
- body-parser
- md5

## Getting Started

Install the dependencies.

```bash
npm install
```

Run the app.

```bash
npm run local
```
